## 介绍

跨越是一个国铁风格的机械动力附属铁路模组。

作为装饰性模组和功能性模组，跨越为机械动力的铁路系统添加了国铁正在使用或已被淘汰的机车、客车、货车、车载和地面设备，并计划根据国铁规范实现一套功能玩法（如调度，驾驶等）。
